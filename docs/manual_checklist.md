# Ручная проверка TeamFinder

## Проекты

- Главная страница открывается по `/projects/list/`.
- Проекты отображаются карточками и отсортированы от новых к старым.
- На карточке видны автор, дата публикации, название, описание, статус и количество участников.
- Авторизованный пользователь может создать проект.
- Автор проекта может редактировать и завершать свой проект.
- Пользователь может присоединиться к чужому проекту и отказаться от участия.

## Необходимые навыки

- На странице проекта есть блок «Необходимые навыки».
- Владелец проекта видит кнопку добавления навыка.
- При вводе текста появляются подсказки из существующих навыков.
- Если точного совпадения нет, можно создать новый навык.
- Добавление и удаление навыка происходит без перезагрузки страницы.
- На главной странице проекты можно фильтровать по навыку.

## Пользователи

- Регистрация создаёт пользователя и выполняет вход.
- Вход работает по email и паролю.
- Публичный профиль показывает контакты и проекты пользователя.
- Владелец профиля может редактировать свои данные.
- Телефон принимается в формате `8XXXXXXXXXX` или `+7XXXXXXXXXX`.
- Ссылка GitHub должна вести на `github.com`.
- Смена пароля доступна авторизованному пользователю.
- Страница пользователей показывает активных пользователей с пагинацией.

## Администрирование

- Администратор может управлять пользователями.
- Администратор может создавать, редактировать и удалять проекты.
- Администратор может управлять навыками.
